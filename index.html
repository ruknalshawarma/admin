Configuration

Connection Status

The Apps Script URL is hardcoded. This panel reads and writes to the Items and All orders sheets.

Sheet Column Reference

The Items sheet must have these 11 columns in exactly this order:

Col A  Category Name     Col B  Item Name
Col C  Description       Col D  Price
Col E  Offer Price       Col F  Catgory Icon
Col G  Item Image        Col H  All Tags
Col I  Category Name (Arabic)
Col J  Item Name (Arabic)
Col K  Description (Arabic)

The All orders sheet must have these 8 columns:

Col A  OrderID    Col B  Name        Col C  Instruction
Col D  Category   Col E  Item        Col F  Qty
Col G  UnitPrice  Col H  Subtotal

The doPost function should handle saveOrder, updateOrderStatus, getOrders actions.
